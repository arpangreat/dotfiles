Welcome to Evernote Web Clipper!

Installing the Evernote Web Clipper will allow you to collect web pages and save them to your Evernote account. To use Web Clipper we need your permission to:

  • Capture webpages to be stored in Evernote
  • Capture urls for the saved webpages

If you decline permissions you will not be able to use Web Clipper. You can disable this service at anytime.

For more information, please see our Privacy Policy.
