174 --HACK:
175 -- Try to prevent bad habits like using the arrow keys for movement. This is
176 -- not the only possible bad habit. For example, holding down the h/j/k/l keys
177 -- for movement, rather than using more efficient movement commands, is also a
178 -- bad habit. The former is enforceable through a .vimrc, while we don't know
179 -- how to prevent the latter.
180 -- Do this in normal mode...
181 vim.cmd('nnoremap <Left>  :echoe "Use h"<CR>')
182 vim.cmd('nnoremap <Right> :echoe "Use l"<CR>')
183 vim.cmd('nnoremap <Up>    :echoe "Use k"<CR>')
184 vim.cmd('nnoremap <Down>  :echoe "Use j"<CR>')
185 -- ...and in insert mode
186 vim.cmd('inoremap <Left>  <ESC>:echoe "Use h"<CR>')
187 vim.cmd('inoremap <Right> <ESC>:echoe "Use l"<CR>')
188 vim.cmd('inoremap <Up>    <ESC>:echoe "Use k"<CR>')
189 vim.cmd('inoremap <Down>  <ESC>:echoe "Use j"<CR>')
